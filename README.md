## Installation

### From Pypi

As simple as doing:

<pre>
pip install django-searchbar
</pre>

### From source

To install django-searchbar from source:

<pre>
git clone https://github.com/arsham/django-searchbar/
cd django-searchbar
python setup.py install
</pre>
